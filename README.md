## **Using 📦**

1. Clone Template

2. Install Packages

```
yarn install
```

3. Start Project

```
yarn dev
```

## **Options ✍️**

1. Check lint

```
yarn lint
```

2. Fix lint

```
yarn lint:fix
```

3. Check prettier

```
yarn prettier
```

4. Fix prettier

```
yarn prettier:fix
```

5. Fix lint and prettier

```
yarn format
```
