## **Using 📦**

1. Clone Template

2. Install Packages

```
yarn install
```

3. Copy .env.example to .env

4. Start dev

```
yarn dev
```

## **Unit test ✍️**

1. Run unit testing

```
yarn test
```

2. Run unit testing and coverage

```
yarn test:coverage
```

## **Options ✍️**

1. Check lint

```
yarn lint
```

3. Fix lint and prettier

```
yarn format
```

5. Graphql Codegen

```
yarn codegen
```
